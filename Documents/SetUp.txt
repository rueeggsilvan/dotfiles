Personal guide to set up my linux system.

- Downgrade to latest stable version.
- sudo pacman -Syu
- Install most necessary packages:
		sudo pacman -S nvidia broadcom-wl
		sudo mhwd -a pci nonfree 0300
- Reboot and check if GPU is runnig correctly (nvidia settings) and wifi is workinf without stick.
